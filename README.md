# NetSpeed

**NetSpeed** — это расширение для Firefox, которое показывает:

* входящую скорость
* исходящую скорость
* пинг
* ваш публичный IP
* страну

Расширение открывается прямо из панели Firefox и позволяет быстро проверить интернет без перехода на отдельный сайт.

---

# Что умеет расширение

* измеряет **входящую скорость** (Download)
* измеряет **исходящую скорость** (Upload)
* показывает **Ping**
* показывает **публичный IP**
* показывает **страну**
* имеет красивый интерфейс со спидометром
* работает как обычное расширение Firefox

---

# Как запустить расширение у себя

Есть **2 способа установки**:

## 1. Временная установка

Подходит, если ты просто хочешь быстро проверить расширение у себя.

### Что важно

Такое расширение:

* работает нормально
* появляется в Firefox
* но **удаляется после перезапуска браузера**

---

## 2. Постоянная установка

Подходит, если ты хочешь, чтобы расширение **оставалось в Firefox навсегда**.

### Что важно

Для обычного Firefox постоянная установка делается через **подписанный `.xpi` файл**.

---

# Вариант 1. Временная установка в Firefox

Это самый простой способ для запуска.

## Шаг 1. Скачай или открой папку проекта

У тебя должна быть папка, в которой лежат файлы:

* `manifest.json`
* `popup.html`
* `popup.js`
* `icon.svg`

---

## Шаг 2. Открой Firefox

Запусти Firefox.

---

## Шаг 3. Открой служебную страницу расширений

Вставь в адресную строку:

```text
about:debugging#/runtime/this-firefox
```

И нажми Enter.

---

## Шаг 4. Нажми кнопку загрузки расширения

На открывшейся странице нажми:

```text
Load Temporary Add-on
```

---

## Шаг 5. Выбери файл расширения

Откроется окно выбора файла.

Зайди в папку проекта и выбери файл:

```text
manifest.json
```

---

## Шаг 6. Готово

После этого расширение загрузится в Firefox.

Обычно:

* иконка появится на панели инструментов
* если иконку не видно, нажми на значок расширений в Firefox и закрепи расширение

---

## Шаг 7. Как пользоваться

1. Нажми на иконку расширения
2. Откроется окно NetSpeed
3. Нажми кнопку проверки скорости
4. Дождись окончания измерения

---

## Важно помнить

Если ты:

* закроешь Firefox полностью
* перезапустишь браузер

то временно установленное расширение исчезнет, и его нужно будет загрузить снова.

---

# Вариант 2. Постоянная установка в свой Firefox

Если ты хочешь, чтобы расширение **не удалялось после перезапуска**, нужен **подписанный `.xpi` файл**.

---

## Что такое `.xpi`

`.xpi` — это установочный файл расширения Firefox.

Примерно как:

* `.exe` для программы
* `.apk` для Android
* `.xpi` для расширения Firefox

---

## Как получить `.xpi`

Есть 2 способа:

### Способ A — через Mozilla Developer Hub

Ты загружаешь архив расширения в Mozilla как **самостоятельно распространяемое расширение** и получаешь подписанный `.xpi`.

### Способ B — через `web-ext`

Можно собрать и подписать расширение через консоль.

---

# Самый понятный способ: через сайт Mozilla

Ниже — самый простой маршрут.

---

## Шаг 1. Подготовь архив расширения

У тебя должны быть эти файлы:

* `manifest.json`
* `popup.html`
* `popup.js`
* `icon.svg`

Теперь нужно создать `.zip` архив.

### Важно

Внутри архива должны лежать **сами файлы**, а не ещё одна лишняя папка сверху.

### Правильно:

```text
netspeed.zip
├─ manifest.json
├─ popup.html
├─ popup.js
├─ icon.svg
```

### Неправильно:

```text
netspeed.zip
└─ netspeed/
   ├─ manifest.json
   ├─ popup.html
   ├─ popup.js
   └─ icon.svg
```

---

## Шаг 2. Зайди в Mozilla Developer Hub

Открой сайт Mozilla Add-ons Developer Hub и войди в свой аккаунт Mozilla.

---

## Шаг 3. Создай новую загрузку расширения

Выбери добавление нового расширения.

Когда появится выбор типа размещения, выбери вариант:

```text
Самостоятельно
```

Это значит, что расширение:

* не выкладывается публично в магазин
* не будет видно всем пользователям
* просто будет подписано Mozilla для установки вручную

---

## Шаг 4. Загрузи ZIP-архив

Загрузи свой архив расширения.

Если всё собрано правильно, Mozilla обработает архив и создаст подписанную версию.

---

## Шаг 5. Скачай подписанный `.xpi`

После обработки открой страницу версии расширения и скачай файл `.xpi`.

Именно этот файл нужен для постоянной установки в Firefox.

---

# Как установить подписанный `.xpi` в Firefox

Теперь у тебя есть подписанный файл, например:

```text
netspeed-1.0.1.xpi
```

Дальше делай так.

---

## Шаг 1. Открой менеджер расширений Firefox

В адресной строке открой:

```text
about:addons
```

---

## Шаг 2. Найди значок шестерёнки

Справа сверху на странице расширений будет значок **шестерёнки**.

Нажми на него.

---

## Шаг 3. Выбери установку из файла

В меню нажми:

```text
Install Add-on From File...
```

---

## Шаг 4. Выбери файл `.xpi`

Выбери свой подписанный `.xpi` файл.

---

## Шаг 5. Подтверди установку

Firefox покажет окно подтверждения.

Нажми кнопку установки.

---

## Шаг 6. Готово

После этого расширение будет установлено как обычное.

Теперь оно:

* останется в браузере
* не исчезнет после перезапуска Firefox
* будет открываться как обычное расширение

---

# Как понять, что всё установилось правильно

После установки:

* расширение появится в списке расширений
* иконка появится в панели Firefox или в меню расширений
* при нажатии откроется окно NetSpeed
* кнопка проверки скорости будет работать

---

# Если иконка не видна

Иногда расширение установлено, но иконка не закреплена.

Что делать:

1. Нажми на значок расширений в правом верхнем углу Firefox
2. Найди `NetSpeed`
3. Закрепи его на панели

# Вариант через консоль (для разработчика)

Если хочешь делать всё через терминал, можно использовать `web-ext`.

## Установка `web-ext`

```bash
npm install -g web-ext
```

## Сборка расширения

```bash
web-ext build
```

## Подпись расширения

```bash
web-ext sign --channel=unlisted
```

После подписи ты получишь подписанный `.xpi`, который потом точно так же устанавливается через:

```text
about:addons -> шестерёнка -> Install Add-on From File...
```

---

# Очень коротко: как установить себе

## Если просто протестировать

1. Открыть Firefox
2. Перейти в `about:debugging#/runtime/this-firefox`
3. Нажать `Load Temporary Add-on`
4. Выбрать `manifest.json`

## Если установить навсегда

1. Собрать ZIP
2. Загрузить его в Mozilla как `Самостоятельно`
3. Скачать подписанный `.xpi`
4. Открыть `about:addons`
5. Нажать шестерёнку
6. Выбрать `Install Add-on From File...`
7. Указать `.xpi`

---

# Privacy

Расширение не использует:

* логины
* пароли
* аккаунты
* пользовательскую авторизацию

Расширение делает только сетевые запросы, которые нужны для:

* измерения входящей скорости
* измерения исходящей скорости
* определения пинга
* определения IP
* определения страны
